# Breast Cancer ML Pipeline

A modular, production-ready ML pipeline for breast cancer detection using data preprocessing, dimensionality reduction (PCA), and machine learning models.

## Features

✅ **Data Loading & Exploration** - Load and analyze breast cancer dataset  
✅ **Data Preprocessing** - Clean, encode, and standardize features  
✅ **Dimensionality Reduction** - PCA to reduce features while retaining 95% variance  
✅ **Model Training** - Random Forest classifier with evaluation metrics  
✅ **Modular Design** - Clean, reusable code structure  
✅ **Fully Tested** - Comprehensive test suite included
